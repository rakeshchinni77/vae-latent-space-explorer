# vae-latent-space-explorer

## Overview
This repository will host a production-grade project for exploring and interpreting the latent space of a Variational Autoencoder (VAE).

## Features
- Structured project layout for experimentation and deployment
- Modular architecture for model development and analysis
- Streamlit-based interactive exploration workflow
- Reproducible local development setup with Python 3.10.11 and a virtual environment

## Project Structure
- app/ - Application entry points and UI components
- app/pages/ - Streamlit pages
- app/utils/ - Shared utilities
- app/assets/ - Static assets
- configs/ - Configuration files
- data/ - Raw and processed datasets
- models/ - Model artifacts and checkpoints
- scripts/ - Training and analysis scripts
- tests/ - Automated tests
- notebooks/ - Experiment notebooks
- results/ - Outputs and reports
- docs/ - Documentation and screenshots

## Installation
1. Create and activate a local virtual environment:
   python -m venv .venv
   .venv\\Scripts\\activate
2. Ensure Python 3.10.11 is being used.
3. Install dependencies as they are added during the project lifecycle.

## Docker
Docker support will be added in a later phase.

## Results
Generated model outputs, analysis reports, and visualizations will be stored in the results/ directory.
